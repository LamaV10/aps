# Arduino Parking Sensor
This is a little project, which uses an Arduino paired with a lcd-display and a Ultrasonic-sensor to imitate a parking sensor.

# Tech 
- Arduino Uno
- LCD 1602 Module (16*2)
- Elegoo Ultrasonic-sensor
